A Java-based application designed to record, track, manage, and resolve citizen complaints related 
to public utilities such as electricity, water supply, gas connections, and sewage services.
This project demonstrates structured software design using entity classes, service logic,
and exception handling to simulate a real municipal complaint tracking system.
